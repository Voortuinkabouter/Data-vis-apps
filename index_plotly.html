SLS Grapher

Plot a Graph!

Select one or more excel files to plot the data from an Excel sheet to plot a line chart. The first column will be plotted as 'x' values and the second column as 'y' values. Press the "Next" if you have selected multiple files and want to view the next graph. You can press the "Save" button to download the plot as '.svg'.
