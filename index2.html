SLS Grapher

Plot a Graph!

Press the "Choose Files" button and select one or more SLS files (exported as '.txt') to plot the fraction (%) and cumulative (%) on logaritmic scale. Press the "Next" and "Previous" after selecting multiplefiles to cycle trough the graph. You can press the "Save" button to download the plot as '.svg' or press "Export" to create single Excel file width: diameter, fraction (%) and cumulative (%) for each file on a separate tab.
